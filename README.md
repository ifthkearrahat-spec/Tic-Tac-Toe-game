Tic Tac Toe (Python)
A simple terminal-based Tic Tac Toe game.

How it works
The player chooses 1 to 9 as block number. The computer chooses randomly. The winner is decided using simple function call.

How to run
Make sure Python is installed. Then simply run the game in python.

Controls
1 to 9 for placing your play in the board.
